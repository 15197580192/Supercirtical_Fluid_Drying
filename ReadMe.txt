（1）使用本计算程序之前，请先在参数文件中确认、修改。
（2）参数包括凝胶柱半径、长度、容器半径，萃取介质流速、初始浓度、等效传质系数、等效扩散系数以及时间步长，空间步长可以不设定，最后是总计算时间、密度分布输出频率、残余溶剂输出频率等。
（3）由于在某些情形下（流速过大、时间步长过大），会导致萃取介质中溶剂浓度小于0，进而导致凝胶柱外表面溶剂浓度小于0，这些不符合物理实际的结果无意义。因而，程序中对凝胶柱中最小、最大溶剂浓度进行搜索、显示，方面必要时修改时间步长、流速等。但消耗时间较多，因而，在程序运行之初，可手动输入控制变量，数值为1，表示需要判断是否出现负浓度，为其他数值，则不进行搜索，可以节省时间。
（4）计算输出结果主要有：萃取介质中溶剂浓度分布，残余溶剂量随时间变化，凝胶柱中溶剂浓度分布。
